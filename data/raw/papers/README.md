# Academic Papers

This directory holds research papers used by DiligenceAgent for academic evidence retrieval.
PDFs are not committed due to copyright restrictions.

The system retrieves papers dynamically via PubMed E-utilities API.
For offline use, place PDFs here and set `USE_MOCK_PROVIDERS=false` in `.env`.

Papers used in the reference run (2026-03-24):
- Jumper et al. (2021). "Highly accurate protein structure prediction with AlphaFold." *Nature*.
- Ingraham et al. (2023). "Illuminating protein space with a programmable generative model." *Nature*.
- Abramson et al. (2024). "Accurate structure prediction of biomolecular interactions with AlphaFold 3." *Nature*.
